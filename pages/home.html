Halo

Ini adalah contoh website dengan penerapan paradigma Progressive Website Application atau PWA. Progressive Website Application merupakan sebuah web dengan kemampuan "menyerupai" aplikasi native baik itu aplikasi mobile maupun aplikasi desktop. Sebuah PWA mampu dapat dipakai walau offline, dapat menampilkan notifikasi, dan dapat di install pada sebuah device. Kelebihan PWA sudah pasti akses aplikasi yang lebih cepat dan beberapa informasi bisa di simpan sementara sehingga website tetap bisa di pakai walau sedang offline

Pembuatan website ini dalam rangka submission pada kelas PWA di dicoding. Untuk selanjutnya, aplikasi ini akan saya simpan pada Github dan dapat di akses di https://pwa-sb1.azharlihan.com. Bagi siapa saja yang menemukan website ini, silahkan gunakan sebagai referensi jika di perlukan.

Tema dari website ini adalah tentang informasi sepak bola. Sumber data didapatkan dari football-data.org. Silahkan mendaftar di situs tersebut dengan basic akun untuk mendapatkan data dengan akun basic.
